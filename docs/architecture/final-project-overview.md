# Final Project Overview

The final project shape for this repository is a **mini Transformer-style optimization lab**.

It should eventually contain:

- reference PyTorch implementations
- selected Triton kernels
- provider dispatch and fallbacks
- correctness tests
- benchmarks and profiling reports
- design tradeoff notes

The final project is intentionally small. The point is not to ship a full training framework; the point is to build something compact enough that you can still explain every important design choice.
